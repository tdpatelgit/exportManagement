Restore from a backup

Upload a .zip you previously downloaded here. This replaces all current data (database and product photos) with the contents of that backup. A snapshot of your current data is saved automatically first, so a restore can be undone.

Only genuine backups made by this app are accepted - if you pick the wrong file by mistake, it's rejected and nothing changes. Older backups are automatically upgraded to the current version on restore.
